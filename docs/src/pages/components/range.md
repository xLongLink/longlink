# Range

`<Range>` is the legacy slider component.

Use it only when the page still depends on the older prop model. Use `<Slider>` for the current runtime slider element.

## Example

```xml
<Range
  label="Budget range"
  description="Legacy range control."
  min="0"
  max="100"
  step="10"
  value="[20,80]"
/>
```
