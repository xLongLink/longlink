LongLink is a unified platform for building and operating applications.

The platform combines a control plane with an applications SDK. The control plane standardizes identity, permissions, storage, execution, and observability. The SDK lets you build application logic on top of those shared systems.

This separation keeps platform concerns in one place. You can focus on application behavior, while LongLink manages governance, isolation, and lifecycle management.

## Goals

- Provide a shared control plane for identity, permissions, storage, execution, and observability
- Let teams build applications with a consistent SDK instead of rebuilding platform services
- Keep application development focused on business logic
- Enforce isolation and governance across applications
